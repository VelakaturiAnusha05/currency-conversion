# Currency Converter A responsive Currency Converter web application developed using HTML, CSS, JavaScript, and live API integration. The application performs real-time currency conversions using current exchange rates. --- ## Technologies Used - HTML - CSS - JavaScript - Live Currency Exchange API --- ## Project Description This project is a web-based Currency Converter that allows users to convert an amount from one currency to another using real-time exchange rates. The application fetches live currency data through an API and dynamically updates the conversion result.
